Education Center Management System

A comprehensive Flutter application for managing educational centers with support for students, teachers, classrooms, sessions, and attendance tracking.

Project Overview

The Education Center Management System is a comprehensive Flutter application designed to streamline operations for educational institutions. It provides tools for managing students, teachers, classrooms, sessions, and attendance tracking with an intuitive interface and powerful features.

Key Features

Student & Teacher Management

Complete profiles with subjects and contact information

Classroom & Session Scheduling

Easy scheduling with pricing information

Attendance Tracking

Per session attendance recording and monitoring

Statistics Dashboard

Comprehensive charts and reports for data analysis

Search & Filter

Advanced search across all entities

Technology Stack

Flutter Cross-platform framework
Provider State management
SQLite (sqflite) Local database
Supabase Auth & cloud sync
FL Chart Data visualization
Intl Date formatting & i18n
